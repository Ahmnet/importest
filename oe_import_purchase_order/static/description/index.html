How to use the module: Import Purchase Order

Overview

The Odoo behavior already has the import function but it is somehow difficult to use especially when you willing to import one2many field or advanced field. For example, to import purchase order with purchase order line it is hard to make it and it is lack of validation on each row of excel too when it cannot find master data.
This is the reason why we build this module to support this need. Meaning that you can import the purchase order with purchase order line by following our template. It will validate all the master data from your file and you can correct into the tree view, and export back for final file excel as well. (See the video demo and user guide as below)

Features
Ability to import the excel and csv file by follow the sample template from our third party.
Ability to check the valid data on each row of the excel and correct in the tree view directly.
Ability to export the tree view back after correction
How it works
Watch on YouTube for demo

To make this module work in your database. You have to follow these steps as below:

Step 1: Purchase this module

Step 2: Copy and paste this module to your folder custom addons

Step 3: Rebuild your server

Step 4: Install app

After install the app, then you will be able to use the feature in this module as the below demonstration:


Step 1: Go to Purchase > Order > Import Purchase Order, then you will see the tree view of import purchase order. You can delete those previous import data and reimport the new one by clicking the button Import.


Step 2: When clicking import, you will see the popup as below screenshot. You can upload your excel file but MUST follow our format (You can download the sample one first). If you have done your excel then click to upload file and hit the import after.


Step 3: After you import successfully, then you will see your data from excel into the tree view, and the tree view will tell you if each row has the problem or no. We need to correct the problem if you prefer to import that row because our import function will not import the row which has problem. Status is to tell you if you already sync or not yet. You can sync as many time as you want but if the purchase order exist, it will merge the order line.


Step 4: Tick Each Row you want to import or you can click all then you will see the sync button


Step 5: When you click Sync butotn, it will show the popup for you to choose if you want to auto confirm purchase order, or just a draft as below screenshot.


Step 6: Click Button Sync Now, then it will import all row you checked into purchase order accordingly as below result:
